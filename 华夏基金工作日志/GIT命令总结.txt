1.安装git后需要需要配置全局的用户名和邮箱
$ git config --global user.name "Your Name" 指定仓库用户名
$ git config --global user.email "email@example.com" 制定仓库用户邮箱
注意git config命令的--global参数，用了这个参数，表示你这台机器上所有的Git仓库都会使用这个配置，当然也可以对某个仓库指定不同的用户名和Email地址。

2.创建git版本库
$ git init 将当前路径设为git版本库，此时目录下多了一个.git文件（不能轻易修改里面的内容）
$ git add <file>/git add --all  添加指定文件到暂存区(stage)/添加所有文件到暂存区
$ git commit -m "xxxx" 提交文件到仓库(-m后面是提交描述)
$ git commit --amend 追加提交

3.版本回退
$ git status 查询当前仓库状态
$ git diff    #是工作区(work dict)和暂存区(stage)的比较
$ git diff --cached    #是暂存区(stage)和分支(master)的比较

$ git log [-pretty=oneline] 显示从最近到最远的提交日志[按照清晰格式显示日志]
$ git reflog 查询每一次操作

在Git中，用HEAD表示当前版本，也就是最新的提交3628164...882e1e0（注意我的提交ID和你的肯定不一样），上一个版本就是HEAD^，上上一个版本就是HEAD^^，当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100。
$ git reset --hard HEAD^ 回退到上一个版本
$ git reset --hard commitId 回退到指定提交编号的版本

$ git checkout -- <file> 让文件回到最近一次git commit或git add时的状态
$ git reset HEAD <file> 把暂存区的修改回退到工作区

git clean 参数

    -n 显示 将要 删除的 文件 和  目录

     -f 删除 文件，-df 删除 文件 和 目录
